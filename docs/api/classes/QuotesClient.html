    Class QuotesClient

    Client for LotR quote endpoints.

    Remarks

    Provides methods to list and retrieve quotes. All methods support filtering, sorting, and pagination via the QueryOptions parameter.

    Access via the LotrClient.quotes property:

    const client = new LotrClient({ apiKey: 'your-api-key' });
    const quotes = await client.quotes.list();

    Example

    // Get all quotes from a specific movie
    const quotes = await client.quotes.list({
      filter: { movie: '5cd95395de30eff6ebccde5b' }
    });

    // Get a specific quote
    const quote = await client.quotes.getById('5cd96e05de30eff6ebccec8a');

    // Search for quotes by text (regex)
    const quotes = await client.quotes.list({
      filter: { dialog: { regex: '/ring/i' } }
    });

        pagination: { limit: 20, page: 1 }
      });

        The quote ID to fetch

      Returns Promise<QuoteWithMovie>

      The quote with a movieDetails property containing the full movie object

      Remarks

      Combines /quote/{id} and /movie/{id} into two sequential calls, replacing the raw movie ID string with a full Movie object under movieDetails. All original Quote fields are preserved.

      Throws

      Throws

      Example

      console.log(result.dialog);               // "You shall not pass!"
      console.log(result.movieDetails.name);    // "The Fellowship of the Ring"
      console.log(result.movieDetails.academyAwardWins); // 4

    search

    • search(term: string): AsyncIterable<Quote>

      Searches all quotes by dialog text using a case-insensitive match.

      Parameters

      • term: string

        The text to search for in quote dialog

      Returns AsyncIterable<Quote>

      Async iterable yielding each matching quote

      Remarks

      An async iterator that applies a case-insensitive regex filter on the dialog field and yields matching quotes across all pages. Equivalent to all({ filter: { dialog: { regex: '/term/i' } } }) but with a simpler API for the common case of text search.

      Throws

      AuthenticationError Thrown when API key is invalid or missing.

      Example

      // Find all quotes mentioning "ring"
      for await (const quote of client.quotes.search('ring')) {
        console.log(quote.dialog);
      }

      // Count quotes without storing them all
      let count = 0;
      for await (const _ of client.quotes.search('precious')) {
        count++;
      }
      console.log(`Found ${count} quotes`);

    all

    • all(
          options?: Omit<QueryOptions<QuoteFilter>, "pagination">,
      ): AsyncIterable<Quote>

      Iterates through all quotes using async pagination.

      Parameters

      • Optionaloptions: Omit<QueryOptions<QuoteFilter>, "pagination">

        Optional query options for filtering and sorting (pagination is handled automatically)

      Returns AsyncIterable<Quote>

      Async iterable yielding each quote

      Remarks

      This method automatically handles pagination, yielding each quote one at a time. It's memory-efficient for large result sets as it doesn't load all records into memory.

      The method uses a maximum page size of 100 for efficiency, fetching multiple pages as needed until all quotes have been yielded.

      Throws

      RateLimitError Thrown when API rate limit is exceeded after all retry attempts.

      Throws

      AuthenticationError Thrown when API key is invalid or missing.

      Example

      // Iterate through all quotes
      for await (const quote of client.quotes.all()) {
        console.log(quote.dialog);
      }

      // With filter for specific movie
      for await (const quote of client.quotes.all({ filter: { movie: 'movie-id' } })) {
        console.log(`"${quote.dialog}"`);
      }

      // Search for quotes by text
      for await (const quote of client.quotes.all({ filter: { dialog: { regex: '/ring/i' } } })) {
        console.log(quote.dialog);
      }

      // Memory efficient — count quotes without storing them all
      let count = 0;
      for await (const quote of client.quotes.all()) {
        if (quote.dialog.includes('ring')) count++;
      }
